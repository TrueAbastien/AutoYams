Detailed Description

Video Window dialog for Capture customization and verification.

Author
Bastien LEGOY
Version
1.0.0
Date
Date
2020/05/15

Allow any user to customize the brightness, contrast, framerate and video scale to allow a better recognition of dices through your webcam / video capturing device.
